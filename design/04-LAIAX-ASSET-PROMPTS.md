## Reglas de loop perfecto (Higgsfield Start & End Frames)

1. **Start frame = End frame** = la misma imagen exacta.
2. El prompt describe un **ciclo cerrado**: la materia se mueve, regresa.
3. Cámara con motion **simétrico**: lo que va, vuelve.
4. Render 1080p mínimo, 24 fps.

## Export para web (ffmpeg)

```bash
# MP4 H.264
ffmpeg -i input.mp4 -c:v libx264 -profile:v high -pix_fmt yuv420p \
  -preset slow -crf 20 -movflags +faststart hero-loop.mp4

# WebM VP9 (mejor compresión)
ffmpeg -i input.mp4 -c:v libvpx-vp9 -b:v 0 -crf 32 -row-mt 1 hero-loop.webm

# Poster desde el frame 1
ffmpeg -i hero-loop.mp4 -frames:v 1 -q:v 2 hero-poster.webp
```
