# LAIAX · Design System Pack

Esta carpeta es el **paso F (Foundation)** del método F.R.A.M.E. para construir el sitio LAIAX como un diseñador profesional, no improvisando en el constructor.

## Cómo se usa

1. Adjuntar `01-LAIAX-SKILL.md` al inicio de cualquier sesión con IA (Claude, GPT, Cursor). Es el "skill" que la mantiene dentro de las reglas LAIAX.
2. Leer `02-LAIAX-BRIEF.md` antes de tomar cualquier decisión creativa.
3. Generar imágenes con los prompts en `04-LAIAX-ASSET-PROMPTS.md` usando **GPT Image 2**.
4. Generar videos con esos mismos prompts usando **Higgsfield Seedance 2.0** o **Kling 3.0** (ver reglas Start & End Frames para loops perfectos).
5. Ensamblar la página pegando `05-LAIAX-ONESHOT.md` en Claude / Cursor / Lovable.
6. Correr el handoff a Claude Code para implementar y desplegar.
